Data Structures and Algorithms Visualization Project

🎯 项目说明

本项目为数据结构与算法课程提供可视化支持，通过SVG图形和流程图帮助理解算法的执行过程。 每个HTML文件都是独立的，可以直接在浏览器中打开查看。

打开主页

🚀 使用方法

# 方法1: 直接打开主页
双击 index.html 文件，在浏览器中打开

# 方法2: 使用本地服务器（推荐）

# 然后在浏览器访问 http://localhost:8000

# 方法3: 直接打开单个可视化文件
双击任意 visualization_*.html 文件

🎓 学习建议

1. 先看可视化，再看代码
通过可视化理解算法的整体思路和执行流程，然后再阅读源代码实现细节。
2. 对照学习
将可视化页面和源代码文件同时打开，对照学习效果更佳。
3. 动手实践
看懂算法后，尝试自己实现一遍，加深理解。
4. 分析复杂度
每个可视化页面都标注了时间和空间复杂度，注意理解和分析。
