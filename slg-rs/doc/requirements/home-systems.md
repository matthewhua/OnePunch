# Home 功能系统需求文档

> 对应 Phase：Phase 10（Home 部分）
> 优先级：P1-P3（按系统分级）
> 预估工期：6-8 周（全部系统）

---

## 一、概述

Home Service 中的各功能系统对应玩家的个人数据和操作。每个系统实现 `PlayerSystem` trait，管理自己的数据加载、存盘、命令处理和事件响应。

---

## 二、P1 核心系统

### 2.1 将领系统（HeroSystem）

#### FR-HS-001：将领基础
- 将领招募：通过抽卡、活动、任务获得
- 将领属性：攻击、防御、智力、统率、速度
- 将领等级：消耗经验升级，属性成长
- 将领品质：白、绿、蓝、紫、橙、红

#### FR-HS-002：将领升星
- 消耗同名将领碎片升星
- 升星提升属性和解锁技能槽
- 最高 5 星

#### FR-HS-003：将领技能
- 每个将领有 4 个技能槽
- 技能升级消耗技能书
- 技能效果影响战斗计算

#### FR-HS-004：将领装备
- 将领可穿戴 6 件装备（武器、头盔、铠甲、靴子、饰品×2）
- 装备提供属性加成
- 装备可更换

#### FR-HS-005：将领出征
- 将领可被派遣出征（行军、采集、驻防）
- 出征中的将领不可进行其他操作
- 将领返回后恢复可用状态

### 2.2 背包系统（BackpackSystem）

#### FR-HS-006：道具管理
- 道具获取：战斗掉落、任务奖励、商店购买、活动奖励
- 道具使用：消耗道具获得效果（加速、资源、经验等）
- 道具堆叠：同类道具堆叠存储
- 背包容量：有上限，可通过 VIP 扩展

#### FR-HS-007：道具合成
- 低级道具合成高级道具
- 合成配方从静态配置读取
- 合成消耗材料和金币

#### FR-HS-008：道具分解
- 高级道具分解为低级材料
- 分解比例从静态配置读取

### 2.3 科技系统（TechSystem）

#### FR-HS-009：科技研究
- 科技树结构（前置科技解锁后续科技）
- 研究消耗资源和时间
- 研究完成后获得永久属性加成
- 同时只能研究一项科技

#### FR-HS-010：科技加速
- 使用道具减少研究剩余时间
- 免费加速（每天 N 次，每次减少 M 分钟）
- 联盟帮助加速

#### FR-HS-011：科技效果
- 科技提供各种加成：
  - 资源产出加成
  - 部队属性加成
  - 建筑速度加成
  - 研究速度加成
  - 行军速度加成
  - 采集速度加成

### 2.4 装备系统（EquipSystem）

#### FR-HS-012：装备获取
- 装备锻造：消耗材料制作装备
- 装备掉落：战斗掉落
- 装备品质：白、绿、蓝、紫、橙

#### FR-HS-013：装备强化
- 消耗金币和强化石提升装备等级
- 强化有成功率（高等级成功率降低）
- 强化失败不降级（保底机制）

#### FR-HS-014：装备镶嵌
- 装备有宝石槽位
- 镶嵌宝石提供额外属性
- 宝石可拆卸

### 2.5 建筑系统（BuildingSystem）

#### FR-HS-015：建筑升级
- 消耗资源和时间升级建筑
- 升级提升建筑效果（产出、容量、解锁功能）
- 同时只能升级一个建筑（可通过 VIP 增加队列）
- 建筑等级受主城等级限制

#### FR-HS-016：建筑类型
| 类型 | 说明 |
|------|------|
| 主城 | 解锁其他建筑等级上限 |
| 兵营 | 训练部队 |
| 农场/伐木场/矿场/采石场 | 资源产出 |
| 仓库 | 资源存储上限 |
| 城墙 | 城防属性 |
| 研究所 | 科技研究 |
| 医院 | 伤兵恢复 |
| 兵工厂 | 装备锻造 |

#### FR-HS-017：建筑加速
- 使用道具减少建筑升级剩余时间
- 免费加速
- 联盟帮助加速

### 2.6 任务系统（MissionSystem）

#### FR-HS-018：任务类型
| 类型 | 说明 |
|------|------|
| 主线任务 | 引导玩家成长，按章节推进 |
| 日常任务 | 每日重置，完成获得活跃度 |
| 成长任务 | 一次性成长目标 |
| 限时任务 | 限时完成的特殊任务 |

#### FR-HS-019：任务进度
- 通过事件系统驱动任务进度
- 任务完成后可领取奖励
- 主线任务完成后解锁下一个任务

#### FR-HS-020：活跃度系统
- 完成日常任务获得活跃度
- 活跃度达到阈值领取额外奖励
- 每日重置

## 三、P2 系统

### 3.1 VIP 系统（VipSystem）

#### FR-HS-021：VIP 等级
- 充值获得 VIP 经验
- VIP 经验累计升级 VIP 等级
- VIP 等级提供各种特权

#### FR-HS-022：VIP 特权
- 建筑队列 +1
- 行军队列 +1
- 资源产出加成
- 背包容量扩展
- 免费加速时间增加
- 专属商店

### 3.2 商店系统（ShopSystem）

#### FR-HS-023：商店类型
| 类型 | 说明 |
|------|------|
| 普通商店 | 金币购买 |
| 钻石商店 | 钻石购买 |
| 联盟商店 | 联盟币购买 |
| VIP 商店 | VIP 专属 |
| 活动商店 | 活动货币购买 |

#### FR-HS-024：商店刷新
- 定时自动刷新（每天/每周）
- 手动刷新（消耗钻石）
- 刷新后商品随机生成

### 3.3 邮件系统（MailSystem）

#### FR-HS-025：邮件功能
- 系统邮件：服务端发送（奖励、公告、战报）
- 玩家邮件：玩家间发送
- 邮件附件：可领取的道具/资源
- 邮件过期：超过 30 天自动删除
- 一键领取：批量领取所有附件

### 3.4 聊天系统（ChatSystem）

#### FR-HS-026：聊天频道
| 频道 | 说明 |
|------|------|
| 世界频道 | 全服可见 |
| 阵营频道 | 同阵营可见 |
| 联盟频道 | 同联盟可见 |
| 私聊 | 一对一 |

#### FR-HS-027：聊天功能
- 文字消息
- 表情
- 坐标分享
- 战报分享
- 发言频率限制
- 敏感词过滤

### 3.5 社交系统（SocialSystem）

#### FR-HS-028：好友系统
- 添加/删除好友
- 好友列表（在线状态、战力）
- 好友上限

#### FR-HS-029：黑名单
- 屏蔽玩家消息
- 黑名单管理

## 五、实现建议

### 5.1 实现顺序
1. 将领系统（战斗的基础）
2. 背包系统（道具流转的基础）
3. 建筑系统（城池发展的基础）
4. 科技系统（属性加成的基础）
5. 装备系统（战力提升）
6. 任务系统（引导和留存）
7. 其他系统按需实现

### 5.2 通用模式
每个系统的实现遵循统一模式：
1. 定义数据结构（对应 proto 消息）
2. 实现 PlayerSystem trait（load/save/dirty）
3. 实现命令处理（handle_command）
4. 实现事件响应（handle_event）
5. 实现 ToFunctionClientBase（客户端数据推送）
6. 添加单元测试
